# bitcoin-playground

## description

A UI that demonstrates bitcoin concepts

## getting started

1. install dependencies:
   `npm install`
2. in a terminal, run test suite:
   `npm test`
3. in another terminal, start app in watch mode:
   `npm start`

## dependencies

node 14.16
npm 7.3

## main open source libraries used

bitcoinjs-lib
ethereum-cryptography
ethers
react
typescript
jest
cypress
